1. 画布与基础交互 (Canvas & Core)
+ 无限画布/视口管理：支持缩放（Zoom）、平移（Pan/空格键拖拽）及自适应大屏分辨率。
+ 智能参考线 (Smart Guides)：拖拽时组件间的对齐吸附、等间距提示。
+ 层级管理 (Layer Management)：类似 Photoshop 的图层面板，支持隐藏、锁定、重命名、拖拽调整 Z-index。
+ 右键菜单：复制、粘贴、置顶、置底、成组（Grouping）及解散。
+ 撤销重做 (Undo/Redo)：基于快照或 Action 队列的状态回溯。

2. 组件库与扩展 (Widget System)
+ 基础图表：深度集成的 ECharts（柱状图、折线图、饼图、散点图等）。
+ 地图/3D 视觉：3D 平面地图、3D 建筑城市（结合 Three.js 或 WebGL）。
+ 媒体组件：图片、视频流（RTSP/HLS）、IFRAME 嵌入。
+ 装饰元素：边框容器、动态线条、标题艺术字、素材库（SVG 矢量图标）。

3. 数据中心 (Data Source Management)
+ 多源对接：
    静态数据：直接在编辑器中输入 JSON。
    API 对接：配置 GET/POST 请求、Header 及参数转发。
    数据库直连：SQL 请求支持（通常需后端代理）。
    实时数据：WebSocket、MQTT 或定时轮询更新。
    过滤器 (Data Filter)：使用 JavaScript 脚本对返回的原始数据进行格式化（Map/Reduce）后再喂给图表。

4. 样式配置与主题 (Style & Theming)
+ 属性编辑面板：精细控制位置（X/Y）、尺寸（W/H）、旋转、不透明度。
+ 主题系统：一键切换全局配色方案（背景色、图表调色盘、文字颜色）。
+ CSS 自定义：允许高级用户为特定组件编写自定义样式片段。

5. 发布与运维 (Publish & Ops)
+ 预览模式：全屏无干扰预览，模拟不同分辨率下的展示效果。
+ 发布管理：生成公开链接或私有部署包。
+ 权限控制：区分设计者、查看者。
+ 多端适配策略：等比缩放（拉伸）、等宽缩放、或全屏铺满。
